# ⏳ Countdown Timer

A simple countdown timer built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* Set a countdown timer
* Live countdown display
* Simple and clean UI
* Responsive design

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Project Structure

```
countdowntimer/
│
├── index.html
├── style.css
└── index.js
```

## ▶️ How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
